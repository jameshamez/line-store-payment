# Demo Flow: LINE Food Ordering & Payment

ใช้ไฟล์นี้ส่งให้ลูกค้าเพื่ออธิบายขั้นตอน Demo ระบบสั่งอาหารผ่าน LINE MINI APP

## ลิงก์เริ่มต้น

Demo Guide: `/demo`

## Flow สำหรับพาลูกค้าดู

1. เปิดหน้า Demo Guide
   - ใช้เป็นหน้าแรกสำหรับดูภาพรวมระบบ
   - มีปุ่มไปหน้าสำคัญทั้งหมด

2. ทดลอง Mini App ฝั่งลูกค้า
   - ลูกค้าเปิดจาก QR ของโต๊ะ
   - ระบบรู้ร้านและโต๊ะจาก URL เช่น `shopId` และ `tableId`

3. เลือกเมนูอาหาร
   - เลือกรายการอาหารและจำนวน
   - ระบบคำนวณยอดรวมอัตโนมัติ

4. ส่งยอดชำระเงิน
   - ระบบสร้าง Order
   - สร้าง QR Code สำหรับชำระเงิน
   - บันทึกสถานะแจ้งเตือน LINE

5. ดูหน้า QR Payment / Order Summary
   - แสดงชื่อร้าน
   - แสดงโต๊ะ
   - แสดงรายการอาหาร
   - แสดงยอดรวม
   - แสดง QR Code ชำระเงิน

6. ดู Dashboard ฝั่งร้าน
   - ร้านเห็นว่าโต๊ะไหนมียอดรอชำระ
   - เห็นยอดรวม
   - เห็นสถานะว่าแจ้ง LINE แล้วหรือยัง

7. ดู Table QR Management
   - แสดง QR Code แยกตามโต๊ะ เช่น A1, A2, B1, VIP
   - ใช้สำหรับพิมพ์/ติดโต๊ะในร้าน

8. ดู Menu / Store Management
   - ตัวอย่างการแก้ชื่อร้าน สาขา โต๊ะ เมนู ราคา และสถานะพร้อมขาย
   - ใน Demo นี้เป็นการโชว์ UX ก่อนเชื่อมฐานข้อมูลจริง

## ข้อความแนบตอนส่งลูกค้า

ส่ง Demo Flow ให้ทดลองดูครับ เริ่มจากหน้า Demo Guide แล้วกดตามลำดับได้เลย ระบบจะจำลองการสั่งอาหารผ่าน LINE MINI APP ตั้งแต่สแกน QR ที่โต๊ะ เลือกเมนู สร้าง QR ชำระเงิน ไปจนถึงฝั่งร้านเห็นยอดแยกตามร้านและโต๊ะบน Dashboard ครับ

หมายเหตุ: Demo นี้ใช้สำหรับดู flow และหน้าตาระบบก่อน ส่วนตอนใช้งานจริงจะเชื่อม LINE Official Account / LIFF, ฐานข้อมูลจริง และระบบยืนยันการชำระเงินจริงตาม payment provider ที่เลือกครับ
